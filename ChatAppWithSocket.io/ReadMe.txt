# Real-Time Chat App with Socket.IO, Node.js & React

A simple real-time chat application built with React (frontend) and Node.js + Express + Socket.IO (backend). Features include:

- Real-time bi-directional messaging.
- Typing indicators ("user is typing...").
- Chat history that loads when a new user connects.
- User name is fixed after entry and used for all messages.
- Simple UI with message timestamps.

---

## Technologies Used

- **Frontend:** React, Socket.IO Client, useRef, useState, useEffect
- **Backend:** Node.js, Express, Socket.IO, CORS
- **Others:** Logger for message logging (optional)

---

## Setup & Run

### Prerequisites

- Node.js (v14+ recommended)
- npm or yarn
