# Library Book Management System

A simple **Library Book Management System** built with **React** for the frontend and **Node.js + Express** for the backend.  
This project demonstrates basic CRUD operations (Create, Read, Update, Delete) on a collection of books.

---

## Features

- View all books in a table
- Add new books with details like name, author, pages, price, and availability
- Edit existing books
- Delete books
- Real-time UI updates using React hooks and fetch API
- Backend persists data in a JSON file (`books.json`)

---

## Tech Stack

- Frontend: React (Functional Components, Hooks)
- Backend: Node.js, Express
- Data Storage: JSON file on server
- Styling: CSS with Flexbox
